# Sistem Pakar Diagnosa WiFi

Sistem pakar berbasis web menggunakan metode Forward Chaining dan Certainty Factor untuk mendiagnosa gangguan jaringan WiFi dan internet rumah.

## Teknologi
- Python
- Flask
- HTML
- CSS

## Fitur
- Diagnosa gangguan jaringan
- Forward Chaining
- Certainty Factor
- User Interface berbasis web

## Cara Menjalankan

```bash
pip install flask
python app.py
